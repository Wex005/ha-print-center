# HA Print Center

A reusable printing framework for Home Assistant.

## Features

- PostScript generation
- IPP printing support
- Base64 document encoding
- Printable Home Assistant reports

## Supported Reports

Coming soon:

- Shopping List
- Weather
- Calendar
- Daily Briefing

## Requirements

- Home Assistant
- Pyscript
- IPP Printing integration
- PostScript-capable printer
